# 网站更新说明

以后每次修改网站，按下面做就可以。

## 最简单的方法

推荐使用 `GitHub Desktop`。

步骤：

1. 修改网站文件，比如 `index.html`
2. 打开 `GitHub Desktop`
3. 在左下角填写本次修改说明
4. 点击 `Commit to main`
5. 点击 `Push origin`
6. 等几十秒到 1 分钟，网站会自动更新到线上

## 你改哪里

目前你最常改的文件是：

- `index.html`

如果以后增加图片或其他资源，再一起放到新建的 `assets` 文件夹里。

## 怎么确认有没有更新成功

提交并推送后：

1. 打开 GitHub 仓库，确认最新提交已经上传
2. 等一会儿再打开你的网站
3. 刷新页面查看最新内容

## 如果你想用命令行

也可以在项目文件夹里执行：

```bash
git add .
git commit -m "修改网站内容"
git push
```

效果和 `GitHub Desktop` 一样，推送后网站会自动发布。

## 记住一句话

以后你改完网页，不需要再手动去 Vercel 发布。  
只要把修改推送到 GitHub，网站就会自动更新。
